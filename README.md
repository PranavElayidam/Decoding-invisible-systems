# Decoding Invisible Systems

This is a personal blog and sandbox for exploring hidden infrastructure, low-level computing setups, and human behavioral patterns.

## Built With

- **Framework:** [Astro](https://astro.build)
- **Styling:** Tailwind CSS
- **Content Engine:** Markdown and MDX files
